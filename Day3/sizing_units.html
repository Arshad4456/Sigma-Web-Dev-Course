Sizing Units

In CSS, there are several units of measurement that can be used to define sizes. Here are some of the most common ones:

  • px (pixels): A fixed unit that represents a single pixel on the screen.
  • em: A relative unit that is based on the font size of the element. 1em is equal to the current font size.
  • rem: Similar to em, but it is based on the root font size (usually the font size of the html element).
  • % (percentage): A relative unit that is based on the parent element's size. For example, 50% would be half of the parent element's size.
  • vh (viewport height): A relative unit that is based on the height of the viewport. 1vh is equal to 1% of the viewport height.
  • vw (viewport width): A relative unit that is based on the width of the viewport. 1vw is equal to 1% of the viewport width.

These units can be used for various CSS properties such as width, height, padding, margin, and font-size.
